# React 프론트엔드 설계 V3

## 1. 아키텍처 의도

### 1.1 전체 플랫폼 구조

```
https://sunghoonyk.duckdns.org/
├── / (auth)                    → 로그인/SSO
├── /platform (platform-frame)  → 플랫폼 메인 프레임 (좌측 네비게이션)
│   ├── /platform/scraper       → 스크래퍼 모듈 (React SPA)
│   ├── /platform/resume        → 이력서 모듈 (추후)
│   └── /platform/portfolio     → 포트폴리오 모듈 (추후)
├── /scraper (API)              → 스크래퍼 백엔드 API
└── /grafana, /prometheus       → 모니터링
```

### 1.2 현재 상태 (임시)

현재 `frontend/web/`은 독립 SPA로 배포됨:
- `https://sunghoonyk.duckdns.org/` → React 프론트 (임시)
- 향후 인증 완료 시 플랫폼 프레임 내부로 이동 예정

## 3. API 연동

### 3.1 사용 API

| 기능 | Method | URL | 설명 |
|------|--------|-----|------|
| 크롤러 목록 | GET | `/docs/crawlers` | 크롤러 + 사이트 설정 |
| 디렉토리 목록 | GET | `/docs/list?rootPath=...` | jstree용 파일 트리 |
| 채용공고 조회 | GET | `/docs/jobs?rootPath=...&path=...&site=...` | 페이징 데이터 |
| 크롤러 실행 | POST | `/crawl-config/{id}/execute` | 비동기 실행 |
| 실행 상태 | GET | `/crawl-logs/config/{id}/recent` | 폴링용 |
| 크롤러 설정 | GET/PUT | `/crawl-config/{id}` | CRUD |

### 3.2 jstree 트리 데이터

```json
[
  {
    "id": "java_daily",
    "text": "java_daily",
    "children": [
      {
        "id": "java_daily/2026-07-16",
        "text": "07-16 (360건)",
        "leaf": true
      },
      {
        "id": "java_daily/2026-07-15",
        "text": "07-15 (290건)",
        "leaf": true
      }
    ]
  }
]
```
