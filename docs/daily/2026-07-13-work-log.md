## 1. 계정 연결 (Account Linking) 기능 구현 (완료)

### 구현 내용
- UserProviderEntity + UserProviderRepository 생성
- CustomOAuth2UserService 계정 연결 로직 추가
- AccountLinkService 생성 (link/unlink/list)
- AuthController에 계정 연결 API 4개 추가
- ErrorCode + messages 업데이트

### API 엔드포인트

| Method | URL | 설명 | 인증 |
|--------|-----|------|------|
| GET | /auth/oauth2/link-check | 이메일로 연결된 프로바이더 확인 | X |
| POST | /auth/oauth2/link | 기존 계정에 프로바이더 연결 | X |
| GET | /auth/oauth2/providers | 내 연결된 프로바이더 목록 | O |
| DELETE | /auth/oauth2/providers/{provider} | 프로바이더 연결 해제 | O |

### DB 변경
- user_providers 테이블 생성 (JPA 자동 생성)
- 기존 사용자 데이터 마이그레이션 완료
