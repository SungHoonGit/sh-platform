## 5. 문제 해결 기록

### 문제 1: OAuth2 엔드포인트 404 에러
- 원인: KAKAO_CLIENT_ID 환경변수 미설정
- 해결: systemd 서비스에 환경변수 추가

### 문제 2: nginx에서 OAuth2 경로 미프록시
- 원인: `/oauth2/`, `/login/` 경로가 nginx에 미설정
- 해결: nginx 설정에 proxy location 추가

### 문제 3: 카카오 KOE006 에러
- 원인: Redirect URI 미등록
- 해결: 카카오 개발자 콘솔에서 Redirect URI 등록 필요
